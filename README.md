kc85sdk
=======

Experimental ASM and C SDK for KC85 home computers.

### Getting Started (OSX)

```bash
> brew install sdl2_gfx
> brew install z80asm
> brew install mame
> brew install sdcc
> cd ~
> git clone git@github.com:floooh/kc85sdk.git kc85sdk
> cd kc85sdk
> ./kc config
[...]
> ./kc test
> ./kc asm hello.s
> ./kc run hello
```
